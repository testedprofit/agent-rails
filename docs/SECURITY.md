# Security Policy

Agent Rails is a lightweight guardrail tool. It is not a complete secret scanner, vulnerability scanner, compliance product, or substitute for human security review.

## Supported Security Use

- Catching obvious secret-looking values before they are committed.
- Keeping risky work visible in pull requests.
- Encouraging gated review before releases, deployments, payments, wallets, signing, MainNet, or package publishing.

## Not Guaranteed

- Complete secret detection.
- Complete vulnerability detection.
- Compliance with any legal, financial, healthcare, or regulated requirement.
- Production readiness without human review.

## Reporting Issues

Open a private security advisory or contact the maintainer if the repository is public and security advisories are enabled. Do not paste real credentials into public issues.
